## 🚨 Solução de Problemas

### Erro de Banco
```
Verificar se MySQL está rodando
Acessar: /health.php
```

### Erro de Permissões
```
Linux/Mac: chmod -R 755 pasta/
Windows: Propriedades → Segurança → Controle Total
```

### Erro de PHP
```
Verificar versão: php -v
Instalar extensões: apt-get install php-pdo php-mysql
```
