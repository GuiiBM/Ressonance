# 🎵 Visão Geral do Sistema Ressonance

## 🔍 O que é o Ressonance?

O **Ressonance** é um sistema de **streaming de música** completo, como um "Spotify pessoal". Você pode:

- 🎵 **Tocar músicas** diretamente no navegador
- 💿 **Organizar álbuns** e artistas
- 🎼 **Criar playlists** personalizadas
- 📱 **Navegar facilmente** entre músicas
- 🔊 **Controlar volume** e progresso
- 🎨 **Ver capas** de álbuns e artistas

## 🎯 Como Funciona (Simples)

```
1. 👤 Você coloca arquivos MP3 numa pasta
   ↓
2. 🗄️ Sistema cria banco de dados automaticamente
   ↓
3. 🌐 Acessa pelo navegador (como qualquer site)
   ↓
4. 🎵 Clica numa música → ela toca!
   ↓
5. 🎛️ Usa controles como Spotify (play, pause, volume)
```

## 🛠️ Tecnologias Utilizadas

### 🌐 **Frontend (Interface)**
- **HTML5** - Estrutura das páginas
- **CSS3** - Visual bonito e responsivo
- **JavaScript** - Interatividade e player
- **Font Awesome** - Ícones bonitos

### ⚙️ **Backend (Servidor)**
- **PHP** - Lógica do servidor
- **MySQL** - Banco de dados
- **Apache/Nginx** - Servidor web

### 🎵 **Áudio**
- **HTML5 Audio** - Player nativo do navegador
- **Range Requests** - Streaming eficiente
- **Múltiplos formatos** - MP3, FLAC, WAV, OGG

## 🏗️ Principais Componentes

### 🎵 **Sistema de Áudio**
- Streaming de música sem download
- Suporte a múltiplos formatos
- Player com controles completos
- Sistema de playlists

### 🖼️ **Sistema de Imagens**
- Capas de álbuns e fotos de artistas
- Correção automática de caminhos
- Otimização e cache

### 🗄️ **Banco de Dados**
- Criação automática na primeira execução
- Organização de artistas, álbuns e músicas
- Relacionamentos inteligentes

### 🛣️ **Sistema de Caminhos**
- Detecção automática de onde está instalado
- Correção automática de URLs quebradas
- Funciona em qualquer servidor

## 🎭 Analogia Simples

Imagine o Ressonance como uma **jukebox moderna**:

- 💿 **Você coloca os discos** (arquivos MP3) numa gaveta (pasta)
- 🤖 **A jukebox se organiza sozinha** (banco de dados automático)
- 📱 **Você usa um controle remoto** (navegador web)
- 🎵 **Escolhe e toca as músicas** (interface amigável)
- 🔊 **Controla tudo facilmente** (volume, pular, pausar)
