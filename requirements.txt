# Requisitos mínimos do Ressonance

## Servidor Web
- Apache 2.4+ ou Nginx 1.18+
- Suporte a .htaccess (Apache) ou configuração equivalente (Nginx)

## PHP
- Versão: 7.4 ou superior
- Extensões obrigatórias:
  * pdo
  * pdo_mysql
  * json
  * mbstring
  * fileinfo (recomendado)

## MySQL/MariaDB
- MySQL 5.7+ ou MariaDB 10.3+
- Usuário com privilégios CREATE DATABASE

## Permissões de Sistema
- Leitura/escrita na pasta do projeto
- Criação de diretórios
- Execução de scripts PHP

## Configurações PHP Recomendadas
- memory_limit = 128M
- upload_max_filesize = 50M
- post_max_size = 50M
- max_execution_time = 300

## Opcional
- mod_rewrite (Apache)
- SSL/HTTPS (produção)
- Backup automático (produção)
